📘 회원 권한 Enum (RoleType)
| 필드명 / 값   | 타입   | 설명                        |
| --------- | ---- | ------------------------- |
| `USER`    | Enum | 일반 사용자 권한. 기본적인 서비스 이용 가능 |
| `MANAGER` | Enum | 관리자 권한. 일부 관리 기능 접근 가능    |
| `ADMIN`   | Enum | 최고 관리자 권한. 모든 기능 접근 가능    |

### 📘 회원 테이블 (MemberEntity)
| 필드명                 | 타입              | 설명                  |
|---------------------|-----------------|---------------------|
| `id`                | Long            | 고유 번호 (PK, 자동 생성) |
| `firstName`         | String          | 이름                  |
| `lastName`          | String          | 성                   |
| `username`          | String          | 사용자 아이디             |
| `password`          | String          | 비밀번호                |
| `role` | RoleType | 회원 권한               |

### 📘 회원 서비스 로직 정의서 (MemberService)
| 함수 이름                   | 매개변수                                                                              | 반환형               | 설명                                                                           | 특징 / 비고                                                                                        |
|-------------------------|-----------------------------------------------------------------------------------|-------------------|------------------------------------------------------------------------------| ---------------------------------------------------------------------------------------------- |
| `findAll`               |                                                                                   | `List<MemberDTO>` | 자료 목록 조회 후 DTO로 변환                                                   | |

| `findByUsername`      | `String username`                                                                 | `MemberDTO`       | 단일 자료 조회 후 DTO로 변환                                                           |                                                        |
| `save`                  | `MemberDTO memberDTO`                                                             | `MemberDTO`       | 신규회원 등록|
